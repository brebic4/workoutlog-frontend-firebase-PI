## 📌 Opis aplikacije

WorkoutLog je web aplikacija za praćenje i upravljanje treninzima.  
Korisnicima omogućuje jednostavan unos, pregled i analizu vlastitih treninga kroz listu i kalendarski prikaz.

Aplikacija podržava autentifikaciju korisnika, upravljanje profilom te napredne funkcionalnosti poput filtriranja treninga, spremanja templateova i prikaza statistike.

---

## ⚙️ Funkcionalnosti

- Registracija i prijava korisnika (Firebase Authentication)
- Upravljanje korisničkim profilom (username, lozinka, tema aplikacije)
- Dodavanje, uređivanje i brisanje treninga
- Filtriranje i sortiranje treninga
- List view i calendar view prikaz treninga
- Pregled treninga po odabranom datumu
- Spremanje treninga kao template
- Korištenje i brisanje templateova
- Dashboard sa statistikama treninga
- Admin panel (pregled svih treninga i globalne statistike)
- Dark/Light tema

---

## 🛠️ Tehnologije

- Vue 3 (Composition API)
- Pinia (state management)
- Firebase (Authentication + Firestore)
- TailwindCSS
